Grab content from an external site.

== Description ==

Simply grabs content from another website and places it into yours without need to use an API or RSS feed.

== Installation ==

1) Install WordPress 3.4.2 or higher

2) Download the following file:
http://downloads.wordpress.org/plugin/grab-itt.1.0.zip

3) Login to WordPress admin, click on Plugins / Add New / Upload, then upload the zip file you just downloaded.

4) Activate the plugin.
